# Lucky Scratch & Spin! 🎉

A single-file, mobile-friendly web game for kids with two modes:

- **Scratch Cards** — scratch a colorful card with your finger to reveal a fun fact, joke, or mini-challenge.
- **Spin Wheel** — spin a colorful wheel to land on a random surprise.

Over 130 kid-friendly prizes covering animal facts, space facts, dinosaurs, ocean life, the human body, silly jokes, and fun mini-challenges.

## Run locally

No build step needed — it's a single HTML file.

```bash
python3 -m http.server 8765
```

Then open `http://localhost:8765` in your browser.
